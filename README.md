# Participant Package

This folder contains everything participants need for the hackathon challenge.

Included files:

- `REEFER_PEAK_LOAD_CHALLENGE.md`: participant-facing challenge brief
- `EVALUATION_AND_WINNER_SELECTION.md`: participant-facing evaluation overview
- `REEFER_DATA_COLUMNS.md`: explanation of the reefer dataset columns
- `START_HERE.md`: quickest way into the package
- `reefer_release.zip`: participant-safe reefer data
- `wetterdaten.zip`: historical weather data
- `target_timestamps.csv`: released public timestamps to predict
- `templates/submission_template.csv`: required submission format
- `starter/reefer_starter_notebook.ipynb`: simple starter notebook

Notes:

- `reefer_release.zip` already contains pseudonymized identifiers
- `target_timestamps.csv` contains only the public leaderboard timestamps
- final scoring labels are not included in this package
- your handed-in code or notebook must be easy for organizers to rerun on the hidden full timestamp list and the complete reefer release data to generate the final full submission
- this folder is the participant-facing version of the challenge pack
